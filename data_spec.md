# Dataset Specification

## Dataset Source
This project uses the VisDrone dataset.

## Dataset Description
VisDrone is a drone-based dataset for object detection tasks.

## Categories

The dataset contains the following 9 categories:

0 pedestrian  
1 person  
2 bicycle  
3 car  
4 van  
5 truck  
6 bus  
7 motor  
8 others  

## Dataset Split

The dataset is divided into:

train  
validation  

## Annotation Format

Annotations are converted to YOLO format:

class x_center y_center width height
